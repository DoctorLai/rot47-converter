# Support

Thanks for using the ROT47 Cipher Tool! Here is how to get help.

## Documentation

- [README](./README.md) — features, installation, and usage.
- [AI-generated wiki (DeepWiki)](https://deepwiki.com/DoctorLai/rot47-converter)
  — a browsable overview of the codebase.

## Questions & Discussions

- Browse existing [issues](https://github.com/DoctorLai/rot47-converter/issues)
  to see if your question has already been answered.
- Open a new issue using the appropriate template for bugs or feature requests.

## Reporting Bugs or Requesting Features

Please use the
[issue templates](https://github.com/DoctorLai/rot47-converter/issues/new/choose):

- **Bug report** — for something that isn't working as expected.
- **Feature request** — for new ideas and improvements.

Include as much detail as possible (steps to reproduce, browser/OS, screenshots).

## Security Issues

Please **do not** file public issues for security vulnerabilities. Follow the
process in [SECURITY.md](./SECURITY.md) instead.
